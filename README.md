## ✨ What is Open Design AI?

Open Design AI is a next-generation AI design engineering platform that combines multi-agent workflows, composable design systems, and a local-first runtime into a single creative operating system.

Whether you're a solo founder, startup team, developer, or design engineer — Open Design AI gives you a fully private and customizable AI design workspace powered by your own API keys and coding agents.

```txt
AI Agents + Design Systems + Prompt Engineering + Local Runtime = Open Design AI
```

Built for:

**Creators · Startups · Developers · Designers · AI Builders**

## ⚡ Quick Start

### Prerequisites

* Node.js 24+
* pnpm 10.33.2+
* At least one AI agent CLI installed

### Installation

```bash
git clone https://github.com/saikirantechy/open-design-ai.git

cd open-design-ai

pnpm install

pnpm tools-dev run web
```

Open the local URL printed in your terminal.

### Desktop App

```bash
pnpm tools-dev run desktop
```

---

## 🔑 BYOK Setup

Open Design AI never stores API keys in the cloud.

Configure providers locally:

* `ANTHROPIC_API_KEY`
* `OPENAI_API_KEY`
* `GOOGLE_GENERATIVE_AI_API_KEY`

Add them inside:

```txt
Settings → AI Providers
```
