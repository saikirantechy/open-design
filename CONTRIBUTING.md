# Contributing to Open Design AI

Thank you for your interest in contributing to Open Design AI — part of the SKT Nexus ecosystem!

## Getting Started

1. Fork the repository at `https://github.com/saikirantechy/open-design-ai`
2. Clone your fork locally
3. Follow the [Quick Start](./README.md#-quick-start) guide
4. Make your changes on a new branch
5. Submit a Pull Request

## Code Style

- TypeScript everywhere
- Use `pnpm` for all package management
- Follow existing file/folder conventions
- Add tests for new features where applicable

## Issues

- Check existing issues before opening a new one
- Use clear, descriptive titles
- Include reproduction steps for bugs

## Questions?

Open a Discussion on GitHub or reach out to [Sai Kiran BK](https://github.com/saikirantechy).
